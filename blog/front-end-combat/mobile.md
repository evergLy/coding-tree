# 移动端网页适配

## 屏幕尺寸

屏幕对角线的长度，一般用英寸来度量

## PC 分辨率

```
1920 * 1080
1366 * 768
```

- 硬件分辨率 物理分辨率 不可改变，固定的
- 软件分辨率 逻辑分辨率 由软件驱动决定

移动端主流设备分辨率

| 手机型号    | 物理分辨率  | 逻辑分辨率 | 比例关系 |
| ----------- | ----------- | ---------- | -------- |
| iPhone6/7/8 | 750 \* 1334 | 375 \* 667 | 2:1      |

## 视口

- 移动端网页宽度默认：`980px`
- PC 端网页宽度和屏幕宽度相同

视口的作用:

```
网页宽度 = 逻辑宽度
```

视口标签

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```

二倍图：750px
